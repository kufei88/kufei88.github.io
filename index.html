1. 什么是maven

maven是项目管理工具，主要用于jar包的管理，项目的构建，生成，发布等

2. maven指令

mvn clean compile    //先清空  再编译
mvn clean test  //先清空  再执行单元测试
mvn clean package  //先清空老的文件   再打包新的文件
mvn clean install  //打包到本地库
mvn clean deploy   //打包发布
mvn archetype:generate   //初始化一个Maven项目
mvn install -Dmaven.test.skip=true    //跳过测试

利用HttpServletResponseWrapper修改Response对象

有时我们可能需要从HttpServletResponse读取输出流，并对输出流做一些修改，但是如果我们直接对原始response进行操作则会造成输出流提前打开或者关闭，导致不能从一次请求中完成对输出流的修改。

所以我们可以通过继承HttpServletResponseWrapper对象来获取原始的输出流，通过PrintWriter将输出流写入到Buffer中。

通过过滤器拦截Response对象，然后就可以对输出流进行操作了。
